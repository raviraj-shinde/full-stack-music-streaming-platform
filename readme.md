This project is a full-stack music streaming system with separate user and admin experiences. Users can play music, browse albums, manage playlists (planned), and stream content smoothly. Admin users can upload songs, manage album metadata, and handle content from a dedicated dashboard.

This repository is actively being updated as features are built and refined.

---

## Tech Stack

| Layer | Technology |
|--------|------------|
| Backend | Spring Boot, Spring Security, JWT, Lombok |
| Frontend | React.js, Tailwind CSS |
| Database | MongoDB |
| Media Storage | Cloudinary (Uploads + CDN) |
| Containerization | Docker, Docker Compose |
| APIs | REST APIs |
